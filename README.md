# study_time_calculator
Lets u, add all ur studying minutes together as hour(s),, lets say ur keeping track of ur study minutes like :- 6,45,32,56,7;        U can just copy all of it and paste it into the programe and it will gives u a totalled  hour value. 


## REQUIRMENTS
1, python installed in the system ,
2, input ur minute numbers separated by just a comma (20,40,34,2,6,1,)

# If u wanna use anything other than a comma ,, just replace the comma in line 7 (in split function ) , with ur desired caracter.
